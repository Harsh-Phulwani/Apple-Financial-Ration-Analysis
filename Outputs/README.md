# Financial Health Summary – Apple Inc.

## Overview
This folder contains the consolidated financial ratio outputs derived from the Python-based analysis. The objective is to present a clear, high-level view of Apple’s financial health using structured ratio groupings.

## File Description
A consolidated dataset containing calculated financial ratios across multiple years, categorized by financial performance area.

## Ratio Groupings

### Profitability
Measures the company’s ability to generate earnings relative to revenue, assets, and equity.
-Gross Margin
-Operating Margin
-Return on Equity
-Return on Assets

### Liquidity
Assesses Apple’s ability to meet short-term obligations.
-Current Ratio
-Quick Ratio

### Efficiency & Capital Utilisation
Analyzes how effectively assets and capital are employed.
-Asset Turnover
-Return on Capital Employed (ROCE)

## Year-over-Year (YoY) Profitability Analysis

In addition to absolute ratio values, Year-over-Year (YoY) percentage changes have been calculated for key profitability ratios to assess trend direction and consistency over time.

### YoY Metrics Included
- Gross Margin YoY Change  
- Operating Margin YoY Change
- Net Margin  
- Return on Equity (ROE) YoY Change  
- Return on Assets (ROA) YoY Change  

### Purpose
YoY analysis helps identify:
- Improvements or deterioration in profitability  
- Stability of margins across economic cycles  
- Management efficiency in sustaining returns  

This approach aligns with standard financial analysis practices used in equity research and corporate finance.
